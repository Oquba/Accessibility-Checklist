Accessibility Resources

Check out these helpful resources to make sure your website is accessible and inclusive for everyone!

  • Alt Text for Images: Ensure all images have descriptive alternative text.
    WebAIM: Alternative Text
  • Semantic HTML: Use proper HTML5 elements to convey meaning and structure.
    MDN: Semantics in HTML
  • Color Contrast: Maintain sufficient contrast between text and background.
    WebAIM: Contrast Checker
  • Don't Use Color Alone: Avoid using color as the only visual means to convey information.
    W3C: Use of Color
  • Keyboard Accessibility: All interactive elements should be usable with a keyboard.
    WebAIM: Keyboard Accessibility
  • Focus Visibility: Ensure focus indicators are visible and distinct.
    W3C: Focus Visible
